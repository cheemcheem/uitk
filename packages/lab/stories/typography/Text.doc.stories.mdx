# Typography

We've created a Text component that contains our key typography styles which can improve the effeiciency when building UI components that have typography styles.
Text component is the base of all typography components. This way any text can benefit of the base functionality like truncation and tooltip.

## Props

<ArgsTable of={Text} />

## Typography components

We've created a set of components that are most commonly used, but it's easy to generate your own by passing the tag you want to be returned into the `elementType` prop.
